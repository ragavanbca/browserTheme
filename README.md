# browserTheme
Browser theme template
Using languages
  - HTML
  - CSS
  - JS
  - Bootstrap 5

How setup

    1. Go to browser setting
    2. Go to " On start Up " menu
    3. Set the URL in "Open a specific page or set of pages"
    4. URL : https://ragavanbca.github.io/browserTheme/
    5. Than Restart the Browser.
